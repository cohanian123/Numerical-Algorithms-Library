# Numerical Algorithms Library

A collection of numerical algorithms implemented in Python (Jupyter Notebooks), 
developed across two graduate-level courses in computational mathematics at Florida State University.
Projects spanned core areas of scientific computing including linear algebra, interpolation, numerical integration, iterative solvers, and root-finding. Each algorithm was analyzed theoretically (time/space complexity, error bounds, convergence rates) and validated empirically through structured experiments on randomly generated and structured test cases.
Each module includes a written report with mathematical derivations, algorithmic 
analysis, and empirical results.

## Modules

| Module | Topics |
|--------|--------|
| Structured Matrix Routines | Matrix-vector and matrix-matrix products for triangular matrices; compressed storage formats |
| LU Factorization | LU decomposition with no pivoting and partial pivoting; forward/backward substitution |
| Iterative Linear Solvers | Richardson's, Steepest Descent, Conjugate Gradient, Jacobi, Gauss-Seidel, SGS; CSR sparse format |
| Root Finding | Regula Falsi, Secant, Newton's (standard & modified), Steffensen's method |
| Polynomial Interpolation | Barycentric forms 1 & 2, Newton interpolation, Chebyshev meshes, Runge's phenomenon |
| Piecewise Interpolation & Splines | Cubic Hermite, interpolatory cubic splines (s″ and B-spline parameterizations) |
| Numerical Quadrature | 6 composite rules + adaptive Midpoint and Trapezoidal methods |
